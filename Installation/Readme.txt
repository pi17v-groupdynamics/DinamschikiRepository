	Для правильной установки программного продукта желательно сперва запустить 
JavaChecker.exe. Данная программа проверит наличие на компьютере JRE 1.8 (в случае 
отсутствия среды выполнения, Вам будет предложено скачать JRE с официального сайта).
	Далее Вы сможете запустить установщик программного продукта (Richelieu 
Cipher.exe) прямо из JavaChecker.
